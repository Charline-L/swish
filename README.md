# Swish
  
> dev front de l'application
  
  
### Pour lancer l'app
  
- npm install 
- ng serve
- sera accessible depuis http://localhost:4200/ par défaut


### Écrans intégrés

- Espace de connexion
- Introduction
- Détection objet
- Paramétrage
- Accueil
- Tampon Badges
- Tampon entrainements
- Single entrainement
- Profil
- Entrainement en cours d'enregistrement
- Fin d'entrainement
